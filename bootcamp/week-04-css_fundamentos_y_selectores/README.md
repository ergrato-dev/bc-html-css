# Semana 04 — CSS Fundamentos y Selectores

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Escribir reglas CSS con la sintaxis correcta (selector, propiedad, valor)
- Usar selectores de tipo, clase, ID, atributo, descendiente y combinadores
- Entender y calcular la especificidad de un selector
- Comprender cómo funciona la cascada y la herencia en CSS
- Aplicar estilos con múltiples hojas de estilo (`<link>`)
- Usar la pestaña Styles de DevTools para depurar CSS

## 📚 Requisitos previos

- Semanas 01–03 completadas: HTML5 semántico, formularios y multimedia
- VS Code con Live Server activo
