# Ebooks y Guías Gratuitas — Semana 10

Recursos de lectura gratuitos para reforzar los temas de tipografía, colores y CSS Custom Properties.

---

## Guías Online (Gratuitas)

### web.dev / Learn CSS
Google ofrece un curso completo de CSS con capítulos dedicados:

- **Colors** — https://web.dev/learn/css/color
  Todos los formatos de color, HSL, oklch, opacidad y consistencia.

- **Typography** — https://web.dev/learn/css/typography
  Propiedades tipográficas, font-face, fuentes variable.

- **CSS Custom Properties** — https://web.dev/learn/css/custom-properties
  Declaración, herencia, var(), fallbacks y casos de uso.
