## 🧠 Conocimiento (30 pts)

### Cuestionario (20 pts)

| # | Pregunta | Pts |
|---|----------|-----|
| 1 | ¿Cuál selector tiene mayor especificidad: `.card` o `div.card`? ¿Por qué? | 4 |
| 2 | ¿Qué diferencia hay entre el combinador espacio (` `) y el `>` en CSS? | 3 |
| 3 | ¿Por qué se recomienda evitar `!important` y cuándo sería justificado? | 4 |
| 4 | Dado: `p { color: red }` y `p { color: blue }` en la misma hoja, ¿qué color se aplica? | 3 |
| 5 | ¿Qué propiedades CSS se heredan por defecto y cuáles no? Da 2 ejemplos de cada tipo. | 3 |
| 6 | ¿Cuál es la diferencia entre `initial`, `inherit` y `unset`? | 3 |

### Identificación visual (10 pts)

El instructor mostrará código CSS con problemas de especificidad o cascada. El estudiante debe identificar qué estilo se aplica finalmente y explicar por qué.
