# Webgrafía — Semana 04: CSS Fundamentos y Selectores

## 📚 Referencias principales

### MDN Web Docs
- [Referencia de selectores CSS](https://developer.mozilla.org/es/docs/Web/CSS/CSS_selectors) — Listado completo de todos los selectores CSS con ejemplos
- [Especificidad](https://developer.mozilla.org/es/docs/Web/CSS/CSS_selectors/Specificity) — Cómo calcula el navegador qué regla aplica
- [Cascada y herencia](https://developer.mozilla.org/es/docs/Learn_web_development/Core/Styling_basics/Cascade_and_inheritance) — Guía completa sobre cómo CSS resuelve conflictos
- [Selectores de atributo](https://developer.mozilla.org/es/docs/Web/CSS/Attribute_selectors) — Sintaxis y ejemplos de `[attr^=]`, `[attr$=]`, `[attr*=]`
- [Pseudo-clases](https://developer.mozilla.org/es/docs/Web/CSS/Pseudo-classes) — Lista completa: `:hover`, `:focus`, `:focus-visible`, `:nth-child`, etc.
- [Combinadores](https://developer.mozilla.org/es/docs/Learn_web_development/Core/Styling_basics/Combinators) — Descendiente, hijo directo, hermano adyacente y hermano general
- [valor inherit](https://developer.mozilla.org/es/docs/Web/CSS/inherit) — Cómo forzar herencia en propiedades que no heredan

### W3C / Especificaciones
- [Selectores Level 4 (W3C)](https://www.w3.org/TR/selectors-4/) — Especificación oficial, incluye `:is()`, `:where()`, `:has()`
- [CSS Cascade (W3C)](https://www.w3.org/TR/css-cascade-5/) — Modelo formal de la cascada

### CSS-Tricks
- [La guía de selectores CSS](https://css-tricks.com/almanac/selectors/) — Almanaque completo con demos interactivos
- [Specifics on CSS Specificity](https://css-tricks.com/specifics-on-css-specificity/) — Artículo profundo sobre especificidad con calculadoras
