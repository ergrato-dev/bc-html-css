## 🎨 Recursos de Animación de Referencia

### Animate.css

- **URL:** [animate.style](https://animate.style/)
- **Para qué sirve:** biblioteca de animaciones CSS listas — úsala como **referencia** para estudiar los @keyframes y entender cómo están construidas.
- **Valor educativo:** inspecciona el código fuente para ver cómo implementan bounce, fade, slide y shake.

### Every Layout — Motion chapter

- **URL:** [every-layout.dev](https://every-layout.dev/)
- **Para qué sirve:** principios de diseño responsivo, incluye consideraciones sobre movimiento.
- **Nota:** El libro base es de pago, pero la mayoría del contenido es gratuito online.
