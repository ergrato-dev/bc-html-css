# Semana 03 — HTML: Formularios y Multimedia

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana serás capaz de:

- Crear formularios HTML accesibles con `<form>`, `<input>`, `<label>`, `<fieldset>` y `<legend>`
- Usar los tipos de input correctos según el dato que se recopila
- Construir tablas HTML semánticas con `<thead>`, `<tbody>`, `<tfoot>`
- Incrustar imágenes, video, audio e iframes de forma correcta
- Asociar `<label>` con su `<input>` usando `for`/`id`
- Implementar validación nativa HTML5 (`required`, `pattern`, `min`, `max`)

## 📚 Requisitos previos

- Semana 02 completada: estructura HTML5 y etiquetas semánticas
- VS Code con Live Server instalado y funcionando

## 🎓 Conceptos clave

`<form>` · `<input>` · `<label>` · `<fieldset>` · `<legend>` · `<textarea>` · `<select>` · `<option>` · `<button>` · `<table>` · `<thead>` · `<tbody>` · `<tfoot>` · `<th>` · `<td>` · `colspan` · `rowspan` · `<video>` · `<audio>` · `<source>` · `<iframe>` · `required` · `pattern`

## ✅ Checklist de verificación

- [ ] Todo `<input>` tiene un `<label>` asociado con `for`/`id`
- [ ] Los `<input>` usan el tipo correcto (`email`, `tel`, `date`, etc.)
- [ ] Las tablas usan `<th scope="...">` para accesibilidad
- [ ] Los videos tienen subtítulos o descripción alternativa
- [ ] Los iframes tienen `title` descriptivo
- [ ] El formulario usa `<fieldset>` + `<legend>` para agrupar campos relacionados
