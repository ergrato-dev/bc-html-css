# Ebooks Gratuitos — Semana 09: Responsive & Media Queries

Libros y guías gratuitas de referencia para diseño responsive y CSS moderno.

---

## 📚 Recursos Gratuitos

### 1. web.dev — Learn Responsive Design

- **URL:** https://web.dev/learn/design/
- **Formato:** Web (capítulos interactivos)
- **Temas relevantes:** Mobile-first, media queries, tipografía fluida, container queries
- **Por qué leerlo:** Guía oficial de Google, actualizada y con demos interactivas incluidas.

---

### 2. Interneting Is Hard — Responsive Design (capítulo gratuito)

- **URL:** https://internetingishard.netlify.app/html-and-css/responsive-design/
- **Formato:** Web
- **Temas relevantes:** Fundamentos responsive con imágenes y diagramas muy claros
- **Por qué leerlo:** Extraordinariamente visual y explicativo; ideal para principiantes absolutos.

---

### 3. CSS Notes for Professionals (GoalKicker)

- **URL:** https://goalkicker.com/CSSBook/
- **Formato:** PDF gratuito
- **Páginas relevantes:** Capítulo de Responsive Design y Media Queries (~pp. 100–130)
- **Por qué leerlo:** Referencia rápida con decenas de ejemplos de código. Útil como cheatsheet.

---

### 4. Every Layout — Free Rudiments

- **URL:** https://every-layout.dev/rudiments/
- **Formato:** Web
- **Temas relevantes:** Layout intrinseco sin media queries usando `clamp()`, `min()`, `max()`, `auto-fit`
- **Por qué leerlo:** Enfoque moderno y diferente; demuestra cómo reducir el uso de media queries con CSS inteligente.

---

### 5. Smashing Book: New Adventures in Web Design (capítulos gratuitos)

- **URL:** https://www.smashingmagazine.com/learning/css/
- **Formato:** Web
- **Temas relevantes:** Artículos técnicos sobre responsive design, custom properties, grid y flexbox
- **Por qué leerlo:** Artículos de profundidad técnica escritos por expertos de la comunidad.
