## 📝 Notas

- **Learn CSS (web.dev)** es el recurso más completo y actualizado para comenzar con CSS moderno. Cubre selectores, cascada y herencia de forma progresiva.
- **CSS Notes for Professionals** es una referencia densa pero útil para buscar sintaxis rápidamente.
- Los libros de A Book Apart son de pago, pero tienen extractos gratuitos muy valiosos sobre diseño responsivo y CSS.
