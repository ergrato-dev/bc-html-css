# Proyecto Semana 03 — Formulario de Registro con Multimedia

## 🎯 Objetivo

Crear una página de registro para una conferencia/evento que incluya: formulario completo y accesible, tabla de precios de entradas, y video/audio de presentación del evento.

## 📋 Descripción

Construirás una página para "DevConf 2026" con:

1. **Hero** — nombre del evento, fecha y video de presentación
2. **Tabla de entradas** — precios por tipo con colspan
3. **Formulario de registro** — datos personales, tipo de entrada, sesiones de interés
4. **Footer** — contacto del evento

## 📁 Estructura de archivos

```
3-proyecto/
├── starter/
│   └── index.html        ← Tu punto de partida (con TODOs)
└── solution/             ← Solo para instructores (oculto)
```

## 🚀 Instrucciones

1. Abre `starter/index.html` en VS Code
2. Implementa cada sección siguiendo los `TODO` comentados
3. Verifica el resultado con Live Server
4. Valida en [validator.w3.org](https://validator.w3.org/)

## 📌 Criterios de evaluación

| Criterio | Puntos |
|----------|--------|
| Estructura HTML5 semántica completa | 10 pts |
| Formulario con fieldset/legend y todos los labels | 25 pts |
| Tipos de input correctos y validación HTML5 | 20 pts |
| Tabla con thead/tbody/tfoot y scope en th | 20 pts |
| Multimedia (video/audio/iframe) con accesibilidad | 15 pts |
| Validación W3C sin errores | 10 pts |
| **Total** | **100 pts** |

## ✅ Checklist de entrega

- [ ] Formulario con al menos 6 campos diferentes de tipo distinto
- [ ] Todos los campos tienen `<label>` vinculado
- [ ] Campos obligatorios con `required`
- [ ] Tabla con estructura completa (thead / tbody / tfoot)
- [ ] Al menos un uso de `colspan` o `rowspan`
- [ ] Video o iframe con `title` y/o texto alternativo
- [ ] Sin errores en W3C Validator
