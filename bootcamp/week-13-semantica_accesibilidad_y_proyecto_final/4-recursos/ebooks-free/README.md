# Ebooks y Lecturas Gratuitas — Semana 13

Recursos de lectura gratuitos para profundizar en HTML5 semántico, accesibilidad y rendimiento web.

---

## 📚 Cursos escritos gratuitos (web.dev / MDN)

### Learn HTML — web.dev
- **URL:** [web.dev/learn/html](https://web.dev/learn/html)
- **Nivel:** Principiante a intermedio
- **Descripción:** Curso completo de HTML escrito por Google. Cubre estructura, semántica, formularios, multimedia y más. Incluye texto alternativo, tablas de datos y accesibilidad integrada.
- **Temas clave para esta semana:** Sección 9 (Forms), Sección 14 (Accessibility)

---

### Learn Accessibility — web.dev
- **URL:** [web.dev/learn/accessibility](https://web.dev/learn/accessibility)
- **Nivel:** Principiante a intermedio
- **Descripción:** Guía oficial de accesibilidad de Google. Cubre el árbol de accesibilidad, ARIA, teclado, color, formularios y testing. Uno de los mejores recursos gratuitos disponibles en español.
- **Temas clave:** ARIA basics, Keyboard focus, Color and contrast, Testing

---

### Learn Performance — web.dev
- **URL:** [web.dev/learn/performance](https://web.dev/learn/performance)
- **Nivel:** Principiante a intermedio
- **Descripción:** Curso de rendimiento web de Google. Explica Core Web Vitals (LCP, CLS, INP), optimización de imágenes, carga de recursos y el impacto del HTML en la performance.

---

### MDN HTML Reference
- **URL:** [developer.mozilla.org/es/docs/Web/HTML](https://developer.mozilla.org/es/docs/Web/HTML)
- **Nivel:** Todos los niveles
- **Descripción:** La referencia más completa para HTML. Incluye compatibilidad de navegadores, ejemplos interactivos y guías sobre semántica y accesibilidad. Disponible en español.

---

## 📖 Libros gratuitos online

### Inclusive Components — Heydon Pickering
- **URL:** [inclusive-components.design](https://inclusive-components.design/)
- **Nivel:** Intermedio
- **Descripción:** Blog/libro gratuito con ejemplos detallados de componentes de UI accesibles: toggles, tarjetas, tablas, tooltips, notificaciones y más. Cada capítulo incluye el proceso de pensamiento completo.
- **Capítulos recomendados para esta semana:**
  - Cards (accesible card markup)
  - Tooltips & Toggletips
  - Notifications (role="alert", aria-live)

---

### The A11Y Project — Checklist
- **URL:** [a11yproject.com/checklist](https://www.a11yproject.com/checklist/)
- **Nivel:** Principiante
- **Descripción:** Lista de verificación de accesibilidad categorizada por: Content, Global code, Keyboard, Images, Headings, Lists, Controls, Tables, Forms, Media, Video. Ideal para usar como checklist antes de una entrega.

---

### CSS-Tricks — Complete Guide to HTML Forms
- **URL:** [css-tricks.com/guides](https://css-tricks.com/guides/)
- **Nivel:** Principiante a intermedio
- **Descripción:** Guías visuales del equipo de CSS-Tricks. Especialmente relevante para accesibilidad de formularios y mejores prácticas de HTML5.
