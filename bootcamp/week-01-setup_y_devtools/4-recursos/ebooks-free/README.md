## 📚 Nota sobre la lectura

Para esta semana no es necesario leer libros completos. Se recomienda:

- **Dive Into HTML5**: sección 1 (historia y estructura de HTML5)
- **HTML & CSS Is Hard**: capítulos 1 y 2 (introducción y configuración)
- **VS Code Tips and Tricks**: sección de atajos de teclado

El tiempo restante dedícalo a la práctica con las herramientas del entorno.
