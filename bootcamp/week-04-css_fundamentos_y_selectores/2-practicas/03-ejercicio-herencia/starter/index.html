Herencia y Reset CSS

Descubre qué propiedades fluyen de padres a hijos.

Paso 1 — Herencia automática

Este párrafo hereda font-family y color del <body>.

  • Los items de lista también heredan tipografía
  • No necesitas declarar font-family en cada elemento

Paso 2 — Reset con box-sizing

Sin reset el navegador aplica márgenes y padding por defecto. Observa la diferencia en el layout.

Esta caja tiene padding y border. Con box-sizing: border-box el padding no agrega tamaño extra.

Paso 3 — inherit en formularios

Los inputs y botones no heredan font-family por defecto en algunos navegadores.

Paso 4 — initial y unset

Este párrafo usa color: initial — su color es el inicial del navegador.

Este párrafo usa margin: unset.
