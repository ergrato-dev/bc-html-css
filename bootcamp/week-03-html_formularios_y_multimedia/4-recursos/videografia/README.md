## 💡 Tips de aprendizaje

- Cuando veas un formulario en tu sitio web favorito, inspecciona su HTML con DevTools
- Observa si usan `<label>`, `<fieldset>`, tipos correctos de input...
- Identifica problemas de accesibilidad (labels sin for, inputs sin label, etc.)
