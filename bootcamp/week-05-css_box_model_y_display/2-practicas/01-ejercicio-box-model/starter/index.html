Paso 2 — content-box vs border-box

Ambas cajas tienen width: 200px y padding: 20px. ¿Cuál mide 200px en pantalla?

content-box

width: 200px + padding: 20px
Mide más de 200px

border-box

width: 200px incluye padding
Mide exactamente 200px
