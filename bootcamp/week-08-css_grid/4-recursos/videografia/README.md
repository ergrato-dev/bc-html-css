# Videografía — CSS Grid

Videos recomendados para reforzar y profundizar CSS Grid.

## 🎬 Videos esenciales

### CSS Grid Tutorial para principiantes — Kevin Powell
- **Duración:** ~30 min
- **Nivel:** Principiante / Intermedio
- **Canal:** Kevin Powell (YouTube)
- **Buscar:** "CSS Grid Tutorial for beginners kevin powell"
- **Por qué:** Explicación visual clara de tracks, líneas, áreas y funciones. El mejor punto de entrada.

### CSS Grid Crash Course — Traversy Media
- **Duración:** ~40 min
- **Nivel:** Principiante
- **Canal:** Traversy Media (YouTube)
- **Buscar:** "CSS Grid crash course traversy media"
- **Por qué:** Ejemplos prácticos desde cero, cubre todos los conceptos del bootcamp.

### CSS Grid in 100 Seconds — Fireship
- **Duración:** ~2 min
- **Nivel:** Referencia rápida
- **Canal:** Fireship (YouTube)
- **Buscar:** "CSS Grid in 100 seconds fireship"
- **Por qué:** Repaso ultra-rápido de los conceptos más importantes.

### Build a Classic Layout FAST in CSS Grid — Morten Rand-Hendriksen
- **Duración:** ~10 min
- **Nivel:** Principiante
- **Canal:** Mozilla Developer (YouTube)
- **Buscar:** "Build a Classic Layout FAST in CSS Grid"
- **Por qué:** Demuestra cómo CSS Grid simplifica layouts que antes requerían hacks complejos.

### Jen Simmons — Layout Land
- **Duración:** Serie (múltiples videos)
- **Nivel:** Principiante / Intermedio
- **Canal:** Jen Simmons (YouTube)
- **Buscar:** "Layout Land Jen Simmons CSS Grid"
- **Por qué:** Demostración de diseños editoriales creativos con CSS Grid puro.

## 🔧 Videos de DevTools

### Inspect CSS Grid with Chrome DevTools
- **Buscar:** "Chrome DevTools CSS Grid inspector"
- **Por qué:** Aprende a usar el Grid Inspector para depurar layouts.

## 💡 Consejo

Aprende lo básico con Kevin Powell o Traversy Media (30-40 min), usa Grid Garden para practicar interactivamente, y completa los ejercicios del bootcamp para consolidar. El Grid Inspector de DevTools es tu mejor aliado para entender el comportamiento de los layouts.
