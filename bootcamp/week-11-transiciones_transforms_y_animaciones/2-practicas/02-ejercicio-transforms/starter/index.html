Transform Playground

Descomenta cada PASO para ver el efecto en tiempo real.

PASO 1 — translate(x, y)

El espacio original queda "reservado" — el elemento se mueve visualmente.

translate(20px, -10px)

PASO 2 — rotate(deg)

Positivo = horario, negativo = antihorario.

rotate(15deg)
rotate(-15deg)

PASO 3 — scale(n)

El espacio del layout NO cambia. Solo crece/encoge visualmente.

scale(1.3)
scale(0.7)

PASO 4 — transform-origin (misma rotación, distinto pivote)

Mismos 20deg, pero el punto de giro cambia todo.

top left

transform-origin: top left

transform-origin: center
