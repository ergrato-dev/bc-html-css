# Ebooks Gratuitos — CSS Grid

Libros y guías gratuitas para profundizar en CSS Grid.

## 📚 Recursos gratuitos

### CSS Notes for Professionals
- **Autor:** GoalKicker.com (compilado de Stack Overflow)
- **URL:** https://goalkicker.com/CSSBook/
- **Formato:** PDF gratuito
- **Capítulo relevante:** Sección CSS Grid Layout
- **Por qué:** Referencia técnica concisa con ejemplos reales extraídos de la comunidad.

### web.dev — Learn CSS (Módulo Grid)
- **Autor:** Chrome DevRel Team (Google)
- **URL:** https://web.dev/learn/css/grid/
- **Formato:** Web interactiva gratuita
- **Por qué:** Explicaciones directas con demos interactivos en el navegador. Actualizado constantemente.

### Grid by Example — Rachel Andrew
- **Autora:** Rachel Andrew (editora de CSS Working Group)
- **URL:** https://gridbyexample.com/
- **Formato:** Web gratuita con ejemplos descargables
- **Por qué:** La colección más completa de patrones CSS Grid con código funcional. Rachel Andrew es una de las principales impulsoras de CSS Grid.

### CSS Grid — The Definitive Guide (O'Reilly Sample)
- **Autores:** Eric Meyer, Estelle Weyl
- **URL:** https://www.oreilly.com/library/view/css-the-definitive/9781098117603/ (algunos capítulos gratuitos)
- **Formato:** Web (capítulos de muestra gratuitos)
- **Por qué:** Referencia técnica exhaustiva de dos expertos reconocidos en CSS.

### The CSS Handbook — Flavio Copes
- **Autor:** Flavio Copes
- **URL:** https://flaviocopes.com/page/css-handbook/ (descarga gratuita registrándose)
- **Formato:** PDF gratuito
- **Capítulo relevante:** CSS Grid
- **Por qué:** Guía completa y actualizada, lenguaje accesible.

## 💡 Recomendación de estudio

Para este bootcamp, el recurso más valioso es **Grid by Example** para ver patrones reales, combinado con el módulo interactivo de **web.dev** para practicar conceptos paso a paso.
