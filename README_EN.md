## 📋 Description

Intensive **13-week (~3 months)** bootcamp focused on mastering modern **HTML5 and CSS3**. Designed to take students from zero to **Junior Frontend Developer**, with emphasis on semantics, accessibility, modern layouts, and responsive design.

### 🎯 Objectives

Upon completion of the bootcamp, students will be able to:

- ✅ Set up the development environment (VS Code, Live Server, DevTools)
- ✅ Inspect, debug, and experiment with HTML/CSS from the browser
- ✅ Write semantic, structured, and accessible HTML5
- ✅ Create forms, tables, and multimedia correctly
- ✅ Master the Box Model and display modes
- ✅ Build layouts with Flexbox and CSS Grid
- ✅ Implement responsive design with Mobile-first and Media Queries
- ✅ Use CSS Custom Properties, web typography, and color palettes
- ✅ Apply CSS transitions, transforms, and animations
- ✅ Use advanced selectors, pseudo-classes, and pseudo-elements
- ✅ Meet accessibility standards (ARIA, a11y) and basic SEO
- ✅ Build a complete responsive portfolio as a final project

### 🎨 Why HTML + CSS First?

> **Strong foundations make the difference** — No shortcuts, no frameworks from day 1.

HTML and CSS are the pillars of any web interface. Before using frameworks like Tailwind, Bootstrap, or React, it's essential to master the browser's native language. This bootcamp focuses exclusively on semantic HTML5 and modern CSS3, building a solid foundation that will make better developers regardless of the stack they use later.

## 🚀 Quick Start

### Prerequisites

- **VS Code** installed with the **Live Server** extension
- **Git** for version control
- Modern browser (Chrome or Firefox recommended)

### 1. Clone the Repository

```bash
git clone https://github.com/ergrato-dev/bc-html-css.git
cd bc-html-css
```

### 2. Install VS Code Extensions

```bash
# Open in VS Code
code .

# Recommended extensions will appear automatically
# Or run: Ctrl+Shift+P → "Extensions: Show Recommended Extensions"
```

### 3. Navigate to Current Week

```bash
cd bootcamp/week-01-setup_y_devtools
```

### 4. Open with Live Server

Open the week's `index.html` file and click **"Go Live"** in VS Code's status bar.

## 📊 Learning Methodology

### Teaching Strategies

- 🎯 **Project-Based Learning (PBL)**
- 🧩 **Deliberate Practice**
- 🔨 **Build & Break**: Build and break layouts to understand behavior
- 🔍 **Inspect & Learn**: Intensive use of browser DevTools
- 🎮 **Live Coding**

### Time Distribution (8h/week)

- **Theory**: 2–2.5 hours
- **Practice**: 3.5–4 hours
- **Project**: 1.5–2 hours

### Evaluation

Each week includes three types of evidence:

1. **Knowledge 🧠** (30%): Theoretical assessments and quizzes
2. **Performance 💪** (40%): Practical exercises in class
3. **Product 📦** (30%): Evaluable deliverables (functional HTML/CSS code)

**Passing criteria**: Minimum 70% in each type of evidence
